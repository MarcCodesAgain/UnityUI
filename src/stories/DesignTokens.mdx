## Typography

UnityUI uses two typefaces:

- **Inter** — the editorial voice. Headings, body text, buttons.
- **JetBrains Mono** — the technical voice. Labels, badges, overlines, form field labels, code.

### Font families

```tsx
fontFamily.base   // 'Inter, system-ui, -apple-system, sans-serif'
fontFamily.mono   // 'JetBrains Mono, ui-monospace, monospace'
```

### Font sizes

### Font weights

```tsx
fontWeight.light      // 300
fontWeight.regular    // 400
fontWeight.medium     // 500
fontWeight.semibold   // 600
fontWeight.bold       // 700
fontWeight.black      // 800   — used for h1/h2 and Display
```

### Letter spacing

```tsx
letterSpacing.tighter  // -0.04em   — Display
letterSpacing.tight    // -0.02em   — h1, h2, h3
letterSpacing.normal   //  0em
letterSpacing.wide     //  0.05em
letterSpacing.wider    //  0.08em
letterSpacing.widest   //  0.14em   — overlines, labels (UPPERCASE)
```

## Borders

```tsx
import { borderWidth } from '@unityui/core';

borderWidth[1]   // '1px'  — inputs, cards, dividers, table cells
borderWidth[2]   // '2px'  — accent lines on focus, sort indicator
```

UnityUI has **no border-radius tokens by default** — all corners are square (`border-radius: 0`). This is intentional and central to the Swiss aesthetic.

## Usage with styled-components

```tsx
import styled from 'styled-components';
import { colors, spacing, fontSize, fontWeight, fontFamily, borderWidth } from '@unityui/core';

const MetricCard = styled.div`
  background: ${colors.bgPage};
  border: ${borderWidth[1]} solid ${colors.borderDefault};
  padding: ${spacing[6]};
`;

const MetricValue = styled.span`
  font-family: ${fontFamily.mono};
  font-size: ${fontSize['4xl']};
  font-weight: ${fontWeight.semibold};
  color: ${colors.textPrimary};
  letter-spacing: -0.02em;
`;

const MetricLabel = styled.span`
  font-family: ${fontFamily.mono};
  font-size: ${fontSize.xs};
  font-weight: ${fontWeight.semibold};
  letter-spacing: ${letterSpacing.widest};
  text-transform: uppercase;
  color: ${colors.textSecondary};
`;
```
